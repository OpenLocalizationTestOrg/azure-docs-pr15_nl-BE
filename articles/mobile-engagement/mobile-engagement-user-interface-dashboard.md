# <a name="how-to-view-and-manage-charts-of-data-about-your-application"></a>Het weergeven en beheren van grafieken van gegevens over de toepassing
Het gedeelte van het Dashboard van de gebruikersinterface kan klanten om aangepaste grafieken om hun tijd te maximaliseren door de exacte informatie ze willen niet op in de sectie analytics te maken. Een dashboard geeft de trends van de toepassing en de actieve gebruiker geteld voor de verschillende versies van uw toepassing gedurende een bepaalde tijd (uur/dag/week/maand/aangepaste). De trends die weergegeven zijn gebaseerd op de laatste 7 dagen.

### <a name="see-also"></a>Zie ook
-  [Troubleshooting Guide - Analytics][Link 21]

## <a name="dashboard"></a>Dashboard
- Met de actie 'Verzenden naar dashboard' te kiezen in het menu versnelling van een grafiek in de sectie Analytics van de gebruikersinterface die u zou willen zien op een dashboard kunt u grafieken toevoegen aan een lege dashboard.
 
## <a name="-add-a-dashboard"></a>+ (Een Dashboard toevoegen)
- Het "+" lint menu-item kunt u een nieuw dashboard dat wordt weergegeven in het lintmenu te maken.
 
## <a name="gear-menu"></a>Menu vistuig
- U kunt het menu versnelling van een dashboard te verwijderen, hernoemen, delen, of schema (e-ontvangst) van het dashboard.
