# <a name="i-cant-sign-up-for-azure"></a>Ik kan niet aanmelden voor Azure

Als u zich niet voor Azure aanmelden, zijn er verschillende dingen die u kunt controleren het probleem op te lossen.

## <a name="no-text-messages-or-calls-during-sign-up-account-verification"></a>Geen SMS-berichten of aanroepen tijdens het aanmelden van verificatie van account 

- Controleer of dat uw telefoonnummer een SMS kan ontvangen.
- Controleer het telefoonnummer dat u hebt ingevoerd, met inbegrip van de selectie in de vervolgkeuzelijst land.
- Zorg ervoor dat uw telefoon tekstberichten (SMS) kunt ontvangen als u 'Bericht verzenden' of telefoongesprekken gebruiken als u het alternatief 'Bel me'.
- Als u een mobiele telefoon gebruikt, zorg dat u beschikt over een goede telefoonverbinding.
- Maximaal vier minuten wachten om het berichtensysteem bezorgen uw tekstcode indien gewenst "Bericht verzenden."
- Wanneer u het SMS-bericht ontvangt, geeft u de code in het tekstvak en klik op de knop verificatie om door te gaan.

### <a name="suggestions"></a>Suggesties

- Als er geen tekstberichten (SMS) in je telefoon, de alternatieve verificatie 'Bel me' methode gebruiken.
- Een ander telefoonnummer gebruiken als de verificatie via telefoon stap niet met behulp van SMS- en lukt 'Bel me' methoden.
- Een VOIP-telefoonnummer kan niet worden gebruikt voor de verificatieprocedure.

>[AZURE.NOTE] U kunt uw voorkeur telefoonnummer later wijzigen door [uw profielgegevens bijwerken](billing-how-to-change-azure-account-profile.md).

## <a name="credit-card-declined-or-not-accepted"></a>Creditcard geweigerd of wordt niet geaccepteerd

Zorg ervoor dat de betalingsmethode die u wanneer je je aanmeldt gebruikt voor Azure activeringen of betalingen wordt ondersteund.

- Virtuele en prepaid creditcards worden niet geaccepteerd.
- Geaccepteerde creditcard kaart providers, is afhankelijk van het land rekening.

### <a name="suggestion"></a>Suggestie

Zie [uw bankpas of creditcard is geweigerd Azure teken omhoog](billing-credit-card-fails-during-azure-sign-up.md)voor veelvoorkomende oorzaken van problemen met aanmelding met een creditcard of bankkaart hebt.

## <a name="cant-activate-new-azure-in-open-subscription"></a>Nieuwe Azure In Open abonnement activeren niet

U moet een geldige OSA sleutel hebben met ten minste een Azure In Open token die is gekoppeld aan een nieuwe Azure In Open licentie te activeren.

### <a name="suggestion"></a>Suggestie

Als u niet een OSA sleutel, contactpersoon een van de Partners van Microsoft in [Microsoft Pinpoint](http://pinpoint.microsoft.com/).

## <a name="cant-activate-azure-free-trial"></a>Gratis proefversie Azure activeren niet

Hebt u een Azure-abonnement in het verleden gebruikt? De Azure gebruiksvoorwaarden van gratis activering van de evaluatiesoftware alleen voor een gebruiker die met Azure is beperkt. Als u een ander type Azure abonnement hebt, kunt u een gratis evaluatieversie niet activeren.

### <a name="suggestion"></a>Suggestie

-  Als u een Azure-abonnement in het verleden hebt geactiveerd en de gratis evaluatieversie activeren mislukt, kunt u het Pay-As-You-Go abonnement. 
-  Controleer of u in aanmerking voor een vergoeding aanbieding bent. Meer informatie op de [pagina Microsoft Azure aanbieding](https://azure.microsoft.com/support/legal/offer-details/). Pensioenregelingen moeten specifieke vereisten.
