| Eigenschap | Beschrijving | Toegestane waarden | Vereist |
| -------- | ----------- | -------------- | -------- |
| recursieve | Geeft aan of de gegevens worden gelezen recursief uit de submappen of alleen uit de opgegeven map. | True, False (standaard)| Nee |
